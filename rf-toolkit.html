Drive Test — What To Look At, In Order
  1. Serving cell (best-server) map — fragmented footprints? Overlaps?
  2. RSRP map — coverage holes (<-110 dBm) and overshoot.
  3. SINR map — interference and pilot pollution (<0 dB outside cell edge = problem).
  4. PCI / mod-3 map — PCI confusion in overlap regions kills CRS quality.
  5. Throughput map — does it match RSRP/SINR? If not, look at scheduling or transport.
  6. Handover events — cluster of failures by source–target PCI pair.
  7. Re-establishments / drops — edge = coverage; mid-cell = UL fail / interferer.

Key Formulas
Link budget structure (downlink)
EIRP = Tx Power + Combining Gain + Antenna Gain − Feeder Loss Thermal Noise (dBm) = −174 + 10·log10(BW in Hz) Rx Sensitivity = Thermal Noise + NF + Required SNR MAPL_DL = EIRP − Rx Sensitivity − (Shadow + Interference + Indoor + Body)
Okumura-Hata (urban, 150–1500 MHz)
L = 69.55 + 26.16·log10(f) − 13.82·log10(hb) − a(hm) + (44.9 − 6.55·log10(hb)) · log10(d) a(hm) = (1.1·log10(f) − 0.7)·hm − (1.56·log10(f) − 0.8) f in MHz, hb / hm in m, d in km
COST-231 Hata (1500–2000 MHz)
L = 46.3 + 33.9·log10(f) − 13.82·log10(hb) − a(hm) + (44.9 − 6.55·log10(hb)) · log10(d) + Cm Cm = 0 dB medium city / 3 dB metropolitan
3GPP UMa NLOS (TR 38.901, sub-6 GHz)
PL_UMa_NLOS = 13.54 + 39.08·log10(d_3D) + 20·log10(fc_GHz) − 0.6·(h_UT − 1.5)
Use this for 5G mid-band planning instead of Hata.
Cell area & site count
Site_area = (3√3 / 2) · R² ≈ 2.6 · R² (3-sector hex) N_sites_coverage = Target_area / Site_area N_sites_capacity = Offered_load / Per_site_throughput N_sites = max(N_coverage, N_capacity)
Antenna tilt
Tilt_total = arctan((h_BS − h_target) / d) + (HPBW_v / 2)
First term aims the beam center at target; second puts target at −3 dB edge.

B(A, N) = (A^N / N!) / Σ(k=0..N) (A^k / k!) Solve iteratively for offered load A given target blocking B.
